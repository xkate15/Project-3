# Covid-19 Visualization
This project is for UT DataCamp group work. 
Our topic is Covid-19 Data Visualization. We have included Map, Line Chart and Pie Chart to visualized the data. 



## Technologies


* MySQL WorkBench
* Flask
* Python
* JavaScript
* HTML5
* CSS
* Leaflets
* eCharts
